# Hotel Area Guide

Osaka Umeda hotel guest area guide — find restaurants, shops, and spots near your hotel.

## Hotels
- Toyoko Inn Osaka Umeda Higashi
- Toyoko Inn Umeda Nakatsu I

## Features
- 5 languages (EN / JA / 繁 / KO / TL)
- Category & keyword search
- Walking distance filter
- Google Maps embed
- Print flyer + QR code generation
- Admin panel
